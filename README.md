# Codex Growth Playbook

A practical, continuously updated playbook for developers who want to get real value from Codex-style coding agents.

## Why this repo

Most "AI coding" repos are either too shallow or too theoretical.
This repo focuses on:

- repeatable prompts that produce production-grade output
- workflows for debugging, refactoring, and shipping
- guardrails for security, tests, and code review
- real examples with before/after diffs

## Who it is for

- solo builders
- indie hackers
- engineering students
- teams adopting coding agents

## Repository structure

- `guides/`: actionable workflows and checklists
- `prompts/`: reusable prompt templates
- `examples/`: small, concrete code transformations
- `tools/`: helper scripts and snippets

## Quick start

1. Read `guides/01-workflow.md`.
2. Pick one template from `prompts/`.
3. Run a small task in your own repo.
4. Track outcome quality using the checklist in `guides/quality-checklist.md`.
